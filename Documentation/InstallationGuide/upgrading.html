Upgrading phpPennyAuction from 2.4.1 to 2.4.2

Upgrading the software from 2.4.1 to 2.4.2 is partially automated but you will need to have a fairly good knowledge of servers and backing up data to continue. For this reason, and because version 2.4 comes with an automated installer, phpPennyAuction recommend that as many clients as possible download the newest version and use that if upgrading is not necessary, although note that installing from new may wipe your previous data.



Important: Make sure to read these instructions IN FULL before attempting to upgrade the software. phpPennyAuction CANNOT and WILL NOT be held responsible for any loss or damage caused by not upgrading properly. If you are unsure how to do this, contact phpPennyAuction Support and do not attempt to upgrade as you may lose all your website data.



Step 1. Make your website offline to prevent it from operating during the upgrade process. Go to your Admin Panel and then under the Site Live option in Settings -> General Settings, disable it (off). Your website will then not allow any activity from your users, and will sign everyone out. Verify that this setting has saved and your website has a 'closed' message.

Step 2. Download/Export all the current phpPennyAuction software files from on your server, to somewhere on your hard drive. You must make a backup of ALL of the files and folders used by phpPennyAuction software, including any custom folders for applications or plugins you've added (blogs etc).

Step 3. Once the files backup is complete, you will then need to backup your database as well. You will need to go to phpMyAdmin and use the Export option to export the entire contents of the database to a safe place. If the database is large, please make sure to save the file format as .gzip in phpMyAdmin so that it does not time out.

Step 4. The next step is to upload all of the software's files and folders from 2.4.2 (the newer edition), to your server. This will overwrite all of the files. You don't need to modify the database at all as this process is automated - it's only the website's files on the web server you're updating.

Step 5. If you've made any customizations to the website such as a custom template (theme), then you'll now need to upload those changes from your backup from Step 2. For example if you edited the template_frog/auctions/view.ctp file and want to upload that to work with 2.4.2, you will need to upload it to the server, from your hard drive copy.

Step 6. Once done, log into your website's Admin Panel and the database upgrade process will begin. You don't need to do anything. If there are errors after 5 minutes, log out of the Admin Panel and log back in. The database upgrade should then be fully complete.

Step 7. If you have any questions or if there are problems upgrading, please submit a ticket at the Support Center. Note that phpPennyAuction Staff can only provide free support for the standard software - if you have problems with any modifications/customizations related to the upgrade process we will need to invoice you for the time to look into those as they are NOT covered by our Free Support.
