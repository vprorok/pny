Installing: Step 3 - Final Steps


 


Step 6) Almost there! You now need to create the following 5 cronjobs in your web host's Control Panel (look for the 'cron' button or something similar). Your hosting provider should provide access to this as standard.  These cron jobs should be set to run every minute, of every hour, of every day.  If you cannot do this from your hosting Control Panel, you may need to contact your host for assistance in setting it up.


          The 5 cron commands you need to set up are:


      curl -s http://www.domain.com/daemons.php?type=bidbutler
      curl -s http://www.domain.com/daemons.php?type=extend
      curl -s http://www.domain.com/daemons.php?type=close
      curl -s http://www.domain.com/dcleaner
      curl -s http://www.domain.com/dwinner

 

*http://www.domain.com/ replaces your own domain name.

NOTE: Some servers do not allow cron jobs to be run every 1 minute. In that case, you can use the included cron execution script, which is at webroot/cron_run.php. Try running this file in your Cron control panel at the lowest (most frequent) interval allowed. You'll need to make sure that the cronTime setting in your config.php file matches that though. So if you're running the cron_run.php file every 20th minute, you MUST set the cronTime setting to '20'.




For advanced users, the full Unix command for crons every minute, would be: * * * * curl -s -o /dev/null http://www.domain.com/XXXX





Step 7) You should now be able to access your website on the location you installed it, from your web browser.  Please go to http://www.yourdomain.com/users/register and fill out the form to sign up. Note that the very first account that you register will be set to an administrator.  Once you have done this, open up the config.php file in your HTML Editor (which is in the /app/config/ folder) and turn the demoMode setting back to false to disable further registrations from becoming admin users.




Installation Complete

That’s it! The manual installation is finished and your site is ready to be used. Lastly, log in to your Admin Panel on your website to get started and click the ‘admin’ button (once logged in as the administrator), in the top menu on your website. If you don't see the 'admin' button when logged in, please repeat step 8 above.
