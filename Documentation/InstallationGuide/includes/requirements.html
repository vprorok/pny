Requirements


  • The exact requirements of your server will depend on many factors, your estimated website traffic being one. While we can offer advice about what hardware would be best to use on your server, there is no one setup that works for all.
  • PHP 5.2.0 (or Higher)
  • Linux Server
  • MySQL 4.2 (or Higher)
  • IonCube Loaders (Available free at http://www.ioncube.com/loaders.php)
  • GD Library (free)
  • Mod_rewrite function.
  • ImageMagick.
phpPennyAuction staff cannot advise you as to whether your server supports these requirements. You will need to verify that with your web host.
