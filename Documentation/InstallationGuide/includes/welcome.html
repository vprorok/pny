This page covers how to use it to install the software within 5 minutes.

Using the Automated Installer

You will need an FTP (File Transfer Protocol) program. If you do not have one you can download FileZilla, which is free FTP program and can be used on most operating systems. You do not need an indepth knowledge of PHP programming to install phpPennyAuction, but a working knowledge of FTP functions is a plus. If you require assistance with anything, please open a ticket at the phpPennyAuction Support Center.

Once FileZilla or your FTP program is installed and open:

 

Step 1. Upload the /Installer/install-data/ and /Installer/setup.php file (which is in the "/installer/" folder of your software download), to your web server, making sure to upload both to the highest folder you can see - probably "/public_html/" or just "/html/". Once the upload is complete, navigate to http://www.yourdomain.com/setup.php in your web browser, and the Installer script should load setup.php and show a welcome screen.


  File/Folder Name Item CHMOD Value
/install-data/ 0755
setup.php 0755

Step 2. Follow the instructions on-screen to complete the set up of your phpPennyAuction software. Once completed, your website will then be ready to use.


Your website should now be ready to use.


Note: If there are any issues with the server set up, files or CHMOD values of your files, you'll be presented with warnings to correct those before proceeding. Any issues that are found by the Installer will normally require you to correct those before installation can continue.
